# MediSyncc
smart, IoT-integrated medication adherence platform built for Hackathon 2026 by team Code Hunters. Featuring a persistent looping alarm, AI-driven health insights, and a secure backend to bridge the gap between patients and caregivers.

MediSyncc
 Revolutionizing Medication Adherence with IoT & AI MediSyncc is a full-stack healthcare synchronization system designed to solve the "Silent Crisis" of medication non-adherence. Current statistics show that 50% of chronic patients do not take medications as prescribed, leading to over 125,000 avoidable deaths annually. MediSyncc bridges this gap by transforming passive reminders into an active, intelligent ecosystem.

 🚀 Key Features
 Persistent Smart Alerts: Multi-channel reminders via high-decibel audio loops and LED indicators that demand user interaction.AI Adherence Tracking: A logic-based engine that analyzes patterns to generate a 0–100% daily adherence score and weekly health reports for doctors.Caregiver Loop: Instant cloud-synced notifications to family members (Nominees) if a dose is missed, preventing potential emergencies.SaaS Business Logic: A built-in subscription model featuring an automated 4-day Premium Trial and tiered feature gating.
 
 🛠️ Tech StackFrontend: 
 HTML5, CSS3 (Glassmorphism UI), JavaScript (ES6+).Backend: PHP 8.x, Node.js.Database: MySQL (Relational mapping for schedules and adherence logs).IoT & Cloud: Arduino, Firebase, and AWS IoT for real-time hardware synchronization.
